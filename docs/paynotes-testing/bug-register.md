# Bug register

Use this file to track blockers discovered while building the PayNote suite.
Production bank logic must not be changed in this track to make tests pass; the
correct response is to document the blocker with evidence and root cause.

## Current status

At least one local live scenario still depends on additional harness/event-flow
alignment before it can be enabled reliably.
